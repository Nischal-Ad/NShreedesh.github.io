E-learning

  • Tech Stack: ReactJS, NodeJs, ExpressJs, MongoDB (MERN stack), Typescript, MaterialUI, Styled-Components
  • Project Link: E-Learning

E-learning is a website developed using MERN stack followed by typescript.
It helps users to access courses online and learn anytime & anywhere they want. It uses many tech stacks such as material ui, ts, and more...
It has features like:-
1. User can create alc and browse courses after they login and view details about courses.
2. If user forget password they can reset it by forgetpassword, they get reset mail in email.
3. User can complete quiz given to them during their learning.
4. User can get certified when the user completes their courses.
5. User can comment on their purchased courses as well as edit and delete them.
6. The purchase courses will be accessable for lifetime.
7. If user deletes their id than all courses comment they have done also be deleted.
8. likewise when courses is deleted all lectures, quiz and user enrolled with it also be deleted.
