JobSeeker

  • Tech Stack: MERN stack, Typescript, ANTDesign, TailwindCSS
  • Project Link: JobSeeker (DEVELOPING)

A simple web application where users can browse for project request and request for jobs online anytime & anywhere they want.

It has features like:-
1. Admin can post jobs offers and projects they want to sell.
2. If user forget password they can reset it by forgetpassword, they get reset mail in email.
3. User can apply for jobs and can also see their applied jobs status.
4. User can also browses many projects and can also purchase them.
5. The purchase projects will be accessable for lifetime and can be downloaded anytime.
6. If user deletes their id than all projects they bought and user status also be deleted.
