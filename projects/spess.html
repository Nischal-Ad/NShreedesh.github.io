SPESS

  • Tech Stack: ReactJs, NodeJs, ExpressJs, MongoDB (MERN stack), Typescript, AntDesign, NextJs
  • Project Link: SPESS (landing page) - SPESS (app)

SPESS is a "Shree Pashupati English Secondary School" website developed using MERN stack & NextJs followed by typescript.

It has features like:-
1. Interactive UI for students to browse.
2. Students can view all the notices and news about school.
3. Students can apply online entrance form for class 11.
4. Admin can publish notices and news for students.
5. Admin can open entrance form and provide admit card online.
